# xray-script基于网络跳跃原版魔改

Xray 一键安装脚本，支持与宝塔面板共存

```shell
wget -N --no-check-certificate https://raw.githubusercontent.com/hadis898/Xray-script/main/xray.sh && bash xray.sh
```
